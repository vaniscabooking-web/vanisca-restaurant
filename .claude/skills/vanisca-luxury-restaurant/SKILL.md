# Vanisca Restaurant — Luxury Website Skill

Vanisca is not a normal restaurant website.

Every page, every animation, every interaction, every transition and every
component must feel handcrafted.

The website must communicate luxury, elegance, premium dining and trust.

Never generate generic restaurant layouts.

The final result should feel comparable to:

- Michelin Star Restaurants
- Awwwards Winners
- Apple Product Pages
- Luxury Hospitality Brands
- Premium Hotels
- High-End Fine Dining Experiences

The objective is simple:

> Build one of the most beautiful restaurant websites ever created.

# Claude Operating Rules

Before changing any code:

1. Inspect project structure.

2. Understand component hierarchy.

3. Check performance impact.

4. Preserve existing business logic.

5. Preserve translations.

6. Preserve API routes.

7. Preserve SEO.

8. Preserve responsiveness.

9. Preserve accessibility.

10. Only improve quality.

Claude must NEVER rewrite working code unnecessarily.

Claude must NEVER break integrations.

Claude must NEVER remove features without approval.

Claude should always prefer enhancement over replacement.
